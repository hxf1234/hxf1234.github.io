简介

workerman 是一个高性能的php Socket即时通讯框架.可以用来迅速搭建聊天室或者广播之类的业务.
官网地址:https://www.workerman.net/workerman

gateway-worker则是基于workerman开发的TCP长连接.
gatewayClient是辅助workerman或者gateway进行用户分组以及向用户发送信息的组件, 还能快速绑定系统uid与clientid.

你好,hexo

发表于 2019-05-27

hexo.io

一.安装

1. 需要先安装git与node.js
2. 找到合适的位置,运行安装命令
    npm install -g hexo-cli
4. 初始化目录
    hexo init
5. 安装所需组件
    npm install
6. 开启hexo服务
    hexo server
如果出现了banner为hexo字样的网页即安装成功

二.卸载

npm uninstall hexo-cli -g

三.将hexo与github连接起来

1. 设置好git与其秘钥
    切换至ssh目录
        $ cd ~/.ssh
    生成秘钥
        $ ssh-keygen t rsa -C "youremail"
    添加秘钥到ssh-agent
        $ eval "$(ssh-agent -s)"
    添加生成的SSH key到ssh-agent
        $ ssh-add ~/.ssh/id_rsa
2. 将设置好的ssh复制后添加到自己的git里
    id_rsa.pub文件中的内容
3. 测试是否成功
    $ ssh -T git@github.com
4. 配置Deployment,在根目录_config.yml,修改repo值
