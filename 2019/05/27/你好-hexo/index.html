你好,hexo

发表于 2019-05-27

hexo.io

一.安装

1. 需要先安装git与node.js
2. 找到合适的位置,运行安装命令
    npm install -g hexo-cli
4. 初始化目录
    hexo init
5. 安装所需组件
    npm install
6. 开启hexo服务
    hexo server
如果出现了banner为hexo字样的网页即安装成功

二.卸载

npm uninstall hexo-cli -g

三.将hexo与github连接起来

1. 设置好git与其秘钥
    切换至ssh目录
        $ cd ~/.ssh
    生成秘钥
        $ ssh-keygen t rsa -C "youremail"
    添加秘钥到ssh-agent
        $ eval "$(ssh-agent -s)"
    添加生成的SSH key到ssh-agent
        $ ssh-add ~/.ssh/id_rsa
2. 将设置好的ssh复制后添加到自己的git里
    id_rsa.pub文件中的内容
3. 测试是否成功
    $ ssh -T git@github.com
4. 配置Deployment,在根目录_config.yml,修改repo值

四. keyword

layout 布局

Front-Matter 是文章最上面 —- 分割的区域

draft 草稿

Scaffold 模板

asset 资源

五. 写

可以使用命令生成文章layout
hexo new [layout] <title> #layout在配置文件中,不写默认post布局
在新建文章时会根据Scaffold内相对应的文件建立文件

六. 标签插件

引用块

quote

代码块

code

还可插入视频,图片,引用文章,引用资源等
文档说明

七.资源文件夹

资源文件管理功能开启,在用new命令创建文章的时候,会自动创建一个与文章名相同的文件夹,可以使用绝对路径引入文件流

_config.yml
post_asset_flder: true

相对路径引用的标签插件

八. 用git部署文件

先安装 hexo-deployer-git

$ npm install hexo-deployer-git --save

在编辑好文件后,使用命令一键部署至服务器

$ hexo d -g
